# LC-VIT Experiment Scripts

Pipeline script-based cho regression đa modal nằm trong thư mục này.

Tài liệu chính:
- `guide.md`: cách chạy từng bước
- `implemented.md`: những gì đã được implement và đã verify
- `plan.md`: kế hoạch triển khai và trạng thái checkbox

Entry points:
- `build_regression_manifest.py`
- `extract_features.py`
- `merge_features.py`
- `train_regression.py`
- `eval_regression.py`

Artifact root:
- `artifacts/`
- `runs/`
